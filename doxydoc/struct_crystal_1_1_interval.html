Detailed Description

Represents vectorless interval of the form [a, b] or (a, b) or any combination of exclusive and inclusive end points.

Static class to generate regular Intervals using common types.

Template Parameters
TAny comparent type

This is a vectorless interval, therefore if end component is larger than start component, the interval will swap the two ends around such that a is always lt; b.
